ربط بـ Google Sheets

ربط الموقع بـ Google Sheets (دليل سريع)

خيارات الربط:

  1. استخدم Google Forms: أنشئ نموذجاً وادمج الناتج (Responses) في Google Sheets. ثم يمكنك تضمين النموذج في صفحاتك.
  2. استخدم Google Apps Script و Web App: لإنشاء واجهة API آمنة لقراءة/كتابة البيانات إلى Sheets.
  3. استخدم Google Sheets API باستخدام OAuth 2.0 — مناسب للتطبيقات المتقدمة.

مثال سريع — تضمين Google Form

1) أنشئ Google Form يحتوي على الحقول المطلوبة.
2) من Form اختر إرسال → تضمين (<iframe>) ثم انسخ الكود وألصقه في صفحة HTML (أو استخدم iframe embed في Google Sites).

<iframe src="https://docs.google.com/forms/d/e/FORM_ID/viewform?embedded=true" width="640" height="800" frameborder="0" marginheight="0" marginwidth="0">جارٍ التحميل…</iframe>

ملاحظة أمان

لا تحفظ مفاتيح API في ملفات HTML عامة. للحلول الآمنة استخدم Apps Script أو واجهة خلفية (server).
